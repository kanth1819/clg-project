FlowTrack Database Setup

Follow these steps to connect your FlowTrack project to MySQL

Database Connection Steps
1 Verify MySQL Installation

Make sure MySQL is installed and running on your system.

mysql --version
2 Check Database Configuration

Verify the database configuration in config/database.php:

define('DB_HOST', 'localhost');
define('DB_USER', 'root');
define('DB_PASS', '');
define('DB_NAME', 'flowtrack');

Update these values if your MySQL setup is different.

3 Test Database Connection

Run the test script to verify the database connection:

Test Database Connection

This will create the database and tables if they don't exist.

4 Migrate Data (Optional)

If you have existing data in localStorage that you want to migrate to MySQL:

  1. Export your localStorage data: Export Data
  2. Download the JSON files and place them in the data directory
  3. Run the migration script: Migrate Data
5 Access the Application

Once the database is set up, you can access the application:

Go to Application
Troubleshooting
Common Issues:
  • Connection Failed: Check that MySQL is running and your credentials are correct.
  • Tables Not Created: Make sure PHP has the PDO and PDO_MySQL extensions enabled.
  • Data Migration Failed: Check that the JSON files are in the correct format and location.

For more detailed instructions, refer to the README.md file.
